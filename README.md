Student Grade Calculator

This project is a desktop application developed in Java using JFrame and Swing,
focused on calculating school grade averages. The system allows student
registration, grade input, and automatic calculation of the final average.

The project was created with a learning purpose, aiming to practice Java,
object-oriented programming concepts, and the development of graphical user
interfaces using buttons and panels to provide a simple and intuitive experience.

Features
- Student registration
- Grade input
- School average calculation
- Dynamic result display
- Empty field validation
- Graphical interface built with JFrame and buttons

Technologies Used
- Java
- Java Swing (JFrame, JPanel, JButton, JTextField)
- NetBeans (IDE)

How to Run the Project

1. Clone the repository:
 git clone https://github.com/Viktor-angelo/Student-Average-Calculator.git
2. Open the project in a Java-compatible IDE (NetBeans, IntelliJ, or Eclipse)
3. Run the main application class

Project Purpose
This project was developed for educational purposes, aiming to strengthen basic
and intermediate Java concepts such as event handling, code organization, and
desktop GUI development.
